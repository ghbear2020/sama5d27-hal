Description: The demonstration program reset WDT at regular interval without provoking any error underflow interrupt. User can choose to generate interrupt(underflow/error) and choose if WDT generates interrupt or resets the system. By default WDT is Counter value is set to 10 sec and Delta counter value is set to 6 sec and fault interrupt is enable and WDT reset disable. WDT values can not be changed or it can not be put to reset within first 4 sec (forbidden window)after it starts. Doing so will generate fault interrupt and it restart. Resetting WDT before it reaches zero and after 4 seconds has past will reset it properly without generating any fault or interrupt. If it does not get re-serviced before 10 seconds it will generate underflow error and depending on whether reset is enabled or not it will reset the system.
